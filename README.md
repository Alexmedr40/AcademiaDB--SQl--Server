AcademiaDB - SQL Server Administration & Development
Este proyecto demuestra habilidades avanzadas en el desarrollo y administración de bases de datos relacionales utilizando SQL Server.

Características Principales:
Lógica de Negocio: Implementación de integridad referencial mediante Triggers y Procedimientos Almacenados para controlar inscripciones y roles.

Seguridad y Auditoría: Configuración de Auditoría de Servidor para rastrear eliminaciones y gestión de accesos restringidos (DCL).

Automatización (DBA): Creación de Jobs de SQL Agent para backups automáticos y planes de mantenimiento para la integridad de datos.

Monitoreo: Consultas avanzadas de diagnóstico de rendimiento utilizando vistas de gestión dinámica (DMVs) para identificar bloqueos y optimizar índices.

Herramientas usadas: SQL Server Management Studio (SSMS), T-SQL.
